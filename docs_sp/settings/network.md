# Network

Manage Wi-Fi connections on your comma device.

## Wi-Fi

The Network panel provides a Wi-Fi scanner that lists available networks. Tap a network to connect, or use the **Scan** button to refresh the list.

### Usage Notes

- A stable Wi-Fi connection is required for software updates and route uploads.
- For Firehose Mode, connect to Wi-Fi with a reliable USB-C power adapter at least once per week.
- Hotspot and tethered connections also work for uploads while driving if your data plan allows it.
